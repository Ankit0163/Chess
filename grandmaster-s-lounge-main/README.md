# ♟️ Chess Game

A simple chess game built using React and TypeScript.

## 🚀 Features
- Interactive chess board
- Move validation
- Game timer
- Move history tracking

## 🛠️ Tech Stack
- React
- TypeScript
- Tailwind CSS

## 📦 Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`

- AI opponent
